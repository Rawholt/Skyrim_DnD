# Уровень навыка зачарования

|Уровень навыка|Возможности|
|---|---|
|1|Простые зачарования, обычные души|
|2|Улучшенные эффекты, большие души|
|3|Редкие эффекты, великие души|
|4|Двойные зачарования|
|5|Даэдрические и артефактные эффекты|

# Общие правила зарядов

## Оружие
Каждая атака расходует заряд.

| Сила эффекта | Расход заряда |
| ------------ | ------------- |
| Слабый       | 1             |
| Средний      | 2             |
| Сильный      | 4             |
| Мастерский   | 6             |

## Броня
Пассивные зачарования не расходуют заряды.

## Перезарядка
Перезарядка требует камень души.

| Камень души | Восстановление заряда |
| ----------- | --------------------- |
| Крошечный   | 25                    |
| Малый       | 50                    |
| Обычный     | 100                   |
| Большой     | 200                   |
| Великий     | 400                   |
| Черный      | 500                   |

# Изучение зачарований
Для изучения эффекта требуется уничтожить магический предмет.

| Редкость эффекта | Сложность                 |
| ---------------- | ------------------------- |
| Обычный          | Автоматически             |
| Редкий           | Проверка Интеллекта СЛ 13 |
| Очень редкий     | Проверка Интеллекта СЛ 16 |
| Артефактный      | Проверка Интеллекта СЛ 20 |

# Пример создания предмета

## Стальной меч огня

Основа:

- Стальной меч
- Камень души: Обычный
- Эффект: Огненный удар

Результат:

- +1d6 урона огнем
- 50 зарядов
- Расход 2 заряда за удар

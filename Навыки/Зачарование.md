# Двойное зачарование

## Правила
Начиная с 5 уровня навыка зачарования можно наложить два эффекта на один предмет.

### Ограничения

| Тип предмета | Максимум эффектов |
| ------------ | ----------------- |
| Оружие       | 2                 |
| Броня        | 2                 |
| Украшение    | 2                 |

### Увеличение расхода

| Количество эффектов | Множитель расхода |
| ------------------- | ----------------- |
| 1                   | ×1                |
| 2                   | ×1.5              |

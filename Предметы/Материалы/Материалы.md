# Особенности брони
| Материал            | Бонусный эффект                                                                 |
| ------------------- | ------------------------------------------------------------------------------- |
| Мех                 | Сопротивление холоду                                                            |
| Кожа                | +1 к проверкам скрытности                                                       |
| Железо              | Преимущество против "Толчок"                                                    |
| Сталь               | +1 КД против немагического оружия                                               |
| Орочья              | +1 к запугиванию                                                                |
| Двемерская          | Преимущество против "Толчок" и "Захват"                                         |
| Эльфийская          | Нет помехи к проверкам скрытности                                               |
| Стеклянная          | +1 к инициативе                                                                 |
| Эбонитовая          | Критические попадания по тебе только на натуральную 20                          |
| Даэдрическая        | Враги в 10 футах делают спас бросок мудрости против страха при первом сближении |
| Драконья костяная   | Сопротивление урону от драконьего дыхания                                       |
| Драконья чешуйчатая | +5 футов скорости                                                               |
| Соловьиная          | Преимущество на проверкам скрытности ночью                                      |
| Сталгрим            | Сопротивление холоду и ледяной магии                                            |
| Хитиновая           | +5 футов скорости при отсутствии щита                                           |
| Нордская резная     | Сопротивление холоду + преимущество против страха                               |
| Фалмерская тяжелая  | -2 входящего урона от дальнобойных атак                                         |
| Фалмерская легкая   | Преимущество на проверкам скрытности в пещерах и подземельях                    |

# Особенности оружия
| Материал     | Бонусный эффект                                    |
| ------------ | -------------------------------------------------- |
| Железо       | Оружие можно чинить в полевых условиях без кузницы |
| Сталь        | +1 урона против целей в тяжелой броне              |
| Орочье       | Двуручное оружие наносит +2 урона                  |
| Двемерское   | Игнорирует бонус КД от щитов                       |
| Эльфийское   | +1 к броскам атаки                                 |
| Стеклянное   | Критическое попадание на 19–20                     |
| Эбонитовое   | +2 к броскам атаки                                 |
| Даэдрическое | +1d6 некротического урона                          |
| Драконье     | +1d6 урона драконам и драконорожденным             |
| Сталгрим     | Ледяные зачарования наносят +2 урона               |
| Серебряное   | +1d6 урона нежити и даэдра                         |
| Хитиновое    | После перемещения на 10 футов: +1 урона            |
| Нордское     | +1 урона холодом                                   |
| Фалмерское   | Цель теряет 5 футов скорости до следующего хода    |
| Костяное     | Критический урон увеличивается на +1 куб оружия    |
